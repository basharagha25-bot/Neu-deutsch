# Neu A2 - German Learning WebApp

A modern, interactive web application designed to help Arabic speakers learn German at the A2 level.

## Features

- **Interactive Reader**: Click on words for instant translations.
- **Vocabulary Manager**: Save words to your personal list.
- **Dashboard**: Track progress across different units.
- **Practice Modes**: Speaking and Writing exercises.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion

## Setup

1. `npm install`
2. `npm run dev`
